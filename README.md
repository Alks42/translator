# What is this?
Simple translator to translate words from different languges and keep them in dictionary using google translation and linque api for translation, dictionaryapi for meanings synonyms etc and pyqt6 for ui.
## Options
You can translate word and edit translation if you, for example, don't want phonetics for the word (just leave the field blank). Or yo–≥ can completly disable it in settings if you want. You can navigate table with up and down arrows, delete words with delete button and expand row for more information about a word by double clicking it. You can also hide left or right part to check yourself.
